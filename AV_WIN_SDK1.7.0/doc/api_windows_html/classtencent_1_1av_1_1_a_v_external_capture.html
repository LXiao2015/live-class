virtual int tencent::av::AVExternalCapture::OnCaptureFrame ( VideoFrame &  frame)
pure virtual

外部视频捕获设备属于视频输入设备。

参数
frame外部输入的视频帧数据。
备注
选中此设备并打开后，App通过此函数输入视频数据，SDK会将此数据代替摄像头采集的数据发送出去。
