int32 tencent::av::VideoFrameDesc::rotate

画面旋转的角度：

  • source_type为VIDEO_SRC_TYPE_CAMERA时，表示视频源为摄像头。 在终端上，摄像头画面是支持旋转的，App需要根据旋转角度调整渲染层的处理，以保证画面的正常显示。
  • source_type为其他值时，rotate恒为0。
